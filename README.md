# Habit Tracker API

A Spring Boot backend project to track daily habits.

## 🚀 Features
- Create Habit
- Update Habit
- Delete Habit
- Track daily completion

## 🛠 Tech Stack
- Java
- Spring Boot
- JPA (Hibernate)
- MySQL

## 📡 API Endpoints
| Method | Endpoint |
|-------|--------|
| POST | /api/habits |
| GET | /api/habits |
| GET | /api/habits/{id} |
| PUT | /api/habits/{id} |
| DELETE | /api/habits/{id} |

## ▶️ How to Run
1. Clone repo
2. Open in IDE
3. Run Spring Boot Application
